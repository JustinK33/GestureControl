# Project Planning

When invoked:
1. Identify the core user value.
2. Define the MVP.
3. Split into milestones.
4. Recommend the next smallest useful step.
5. Flag what is demo-worthy and what is resume-worthy.

Output sections:
- Goal
- MVP
- Milestones
- Risks
- Next task
