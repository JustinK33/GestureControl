# Project: Hand Gesture Control

## Purpose
This project is a real-time computer vision app using OpenCV and MediaPipe Tasks API.
Current capabilities:
- webcam input
- multi-face detection
- multi-hand landmark drawing
- hand connection drawing

Planned direction:
- gesture-based computer control
- cursor movement with index finger
- pinch-to-click
- later: scroll and gesture shortcuts
